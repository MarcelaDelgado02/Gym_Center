# Gym_Center
Administración y gestión de las operaciones de un gimnasio  
## Estructura del proyecto

- **app/config/** → Contiene la configuración general del sistema.  
- **app/controllers/** → Define la lógica que responde a las acciones del usuario.  
- **app/middleware/** → Procesa solicitudes antes de llegar al controlador (ej. validaciones).  
- **app/models/** → Gestiona las entidades y el acceso a la base de datos.  
- **app/views/** → Plantillas que generan la interfaz visual de la aplicación.  

- **public/index.php** → Punto de entrada principal que redirige las peticiones.  

- **storage/progresoFisico/** → Almacena archivos o datos relacionados con el progreso físico.  

- **README.md** → Documento con instrucciones y guía del proyecto.
